
# Template de Prompt para Criar uma Rota de API

Crie a rota `PUT /workout-plans/{id(uuid)}/days/{id(uuid)}/sessions/:id(uuid)`

## Descrição

Ela atualiza uma sessão de treino específica.

## Autenticação

- Rota protegida.
- Usuário precisa ser o dono do plano de treino recebido.

## Request

```ts
interface Body {
    completedAt: string; // ISO String (2026-02-25T19:46:55:7542)
}
```

```ts
interface Params {}
```

```ts
interface Query {}
```
